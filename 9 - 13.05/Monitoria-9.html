  1. Introduo

Vamos supor que se deseja conhecer um parmetro de certa caracterstica de uma populao representada por uma varivel aleatria X com funo densidade f(x;). Na estimao pontual, o valor de alguma estatstica t(X1,X2,,Xn) estima o parmetro desconhecido .

Exemplo: Suponha uma urna que contm bolas pretas e brancas e que a razo entre elas de 3/1, mas no sabemos se h mais bolas pretas ou brancas. Assim, a probabilidade de retirar uma bola preta 1/4 ou 3/4.

# Parmetros do problema
set.seed(123)
num_simulacoes <- 20
proporcao_real_bolas_pretas <- 0.75
tamanho_amostra <- 12

# Simulao e estimao da proporo de bolas pretas na urna
proporcoes_estimadas <- replicate(num_simulacoes, {
  amostra <- sample(c("preta", "branca"), size = tamanho_amostra, replace = TRUE, prob = c(proporcao_real_bolas_pretas, 1 - proporcao_real_bolas_pretas))
  mean(amostra == "preta")
})

# Histograma das propores estimadas
hist(proporcoes_estimadas, breaks = 40, main = "Estimao da proporo de bolas pretas na urna", xlab = "Proporo estimada", ylab = "Frequncia")

# Mdia das propores estimadas
mean(proporcoes_estimadas)
## [1] 0.7708333

OBS: Quanto maior a amostra, mais prxima da mdia real se torna a mdia calculada.

  1. Funo de densidade de probabilidade (pdf) conjunta

    Dizemos que X e Y so variveis aleatrias independentes se a funo densidade conjunta for o produto das densidades individuais, ou seja, f(x, y) = fX(x).fY(y)

library(ggplot2)

# Parmetros das distribuies de X e Y
media_X <- 0
desvio_X <- 1
media_Y <- 0
desvio_Y <- 2

# Valores de x e y
x_vals <- seq(-3, 3, length.out = 100)
y_vals <- seq(-5, 5, length.out = 100)

# Densidades individuais para X e Y
dens_X <- dnorm(x_vals, mean = media_X, sd = desvio_X)
dens_Y <- dnorm(y_vals, mean = media_Y, sd = desvio_Y)
par(mfrow = c(1, 2))
plot(x_vals, dens_X, type = 'l', main = 'Densidade de X', xlab = 'X', ylab = 'Densidade', col = 'blue')
plot(y_vals, dens_Y, type = 'l', main = 'Densidade de Y', xlab = 'Y', ylab = 'Densidade', col = 'blue')

# Densidade conjunta como o produto das densidades individuais
dens_conjunta <- outer(dens_X, dens_Y, "*")
contour(x_vals, y_vals, dens_conjunta, main = 'Dens. Conj. Produto', xlab = 'X', ylab = 'Y', col = rainbow(10))

# Densidade conjunta usando a funo de densidade conjunta
dens_conjunta_calculada <- outer(dnorm(x_vals, media_X, desvio_X), dnorm(y_vals, media_Y, desvio_Y), "*")
contour(x_vals, y_vals, dens_conjunta_calculada, main = 'Dens. Conj. Calculada', xlab = 'X', ylab = 'Y', col = rainbow(10))

  1. Funo de Verossimilhana

    Para uma amostra aleatria (y1, , yn), a pdf conjunta vista como uma funo do vetor de parmetros desconhecidos denominada de funo de verossimilhana.

    A funo de verossimilhana L(|Y) representa a probabilidade dos dados observados Y como uma funo dos parmetros . L(Y)=f(Y). Em muitos casos, mais conveniente trabalhar com a log-verossimilhana.

O objetivo da funo de verossimilhana avaliar como a probabilidade dos dados varia com diferentes valores do parmetro .

O estimador consistente e assintoticamente eficiente.

  1. Maximizao da Verossimilhana

4.1. Como funciona a maximizao da Verossimilhana?

O Mtodo da MV encontra o valor de que o mais verossmil possvel tendo em vista os dados mo. Ou seja, o estimador maximiza a probabilidade de gerar os dados que realmente temos.

A maximizao da verossimilhana costuma ser resolver uma otimizao para encontrar os valores de que maximizam a funo de verossimilhana. A resoluo do problema consiste em tentar valores para o parmetro at que um critrio de convergncia seja atendido.

  1. Funo de Verossimilhana concentrada

uma verso modificada da funo de verossimilhana que se concentra apenas nos parmetros de interesse da funo de verossimilhana. Vantagem: facilidade de otimizar. Costuma-se realizar uma anlise de sensibilidade, investigando-se como a funo de verossimilhana e as estimativas dos parmetros mudam quando os parmetros so fixados ou modificados.

# Dados de exemplo
set.seed(123)
x <- runif(1000, 0, 10)
beta0 <- 2
beta1 <- 1.5
sigma <- 2
epsilon <- rnorm(1000, 0, sigma)
y <- beta0 + beta1 * x + epsilon

# Definir a funo de verossimilhana concentrada
funcao_verossimilhanca <- function(beta1, y, x, sigma) {
  soma <- sum((y - (beta0 + beta1 * x))^2) / (2 * sigma^2)
  return(-soma) 
}

# Estimar beta1 por otimizao com mtodo "Brent"
resultado <- optimize(f = funcao_verossimilhanca, interval = c(-10, 10), y = y, x = x, sigma = sigma, maximum = TRUE)

# Resultado beta1
resultado$maximum
## [1] 1.500678

6.1. Teste da Razo da Verossimilhana (LR)

Depois de estimar um modelo restrito (parmetros ~) e um irrestrito (parmetros ^) e testamos a H0: h(~) = 0. Se a restrio for verdadeira, o valor da funo de verossimilhana avaliada em ~ e ^ devem estar prximos, indicando que a restrio faz sentido.

library(lmtest)
## Carregando pacotes exigidos: zoo

# Dados de exemplo

x2 <- rnorm(1000, mean = 9, sd = 3)
y <- 2 + 0.5 * x1 + 0.2 * x2 + rnorm(1000, mean = 0, sd = 1)

# Modelos
modelo_irrestrito <- glm(y ~ x1 + x2, family = gaussian(link = "identity"))
modelo_restrito <- glm(y ~ 1, family = gaussian(link = "identity"))

# Teste LR entre modelos
teste_lr <- lrtest(modelo_restrito, modelo_irrestrito)
